最終更新日：2026年7月11日

📌 Instant.gg はユーザーの個人情報を収集・保存しません。入力されたサモナー名は Riot Games API への問い合わせにのみ使用され、サーバーに保存されません。

1. 本サービスについて

Instant.gg は League of Legends のサモナー情報を検索・表示するサービスです。ユーザーが入力したサモナー名を Riot Games API に問い合わせ、その結果を表示するのみで、データの収集・蓄積は行いません。

2. ブラウザのローカルストレージ

本サービスは表示速度向上のため、チャンピオン情報などのゲームデータをユーザーのブラウザ（ローカルストレージ）にキャッシュします。このデータに個人情報は含まれず、サーバーには送信されません。ブラウザの設定からいつでも削除できます。

本サービスは Riot Games の公式サービスではなく、Riot Games, Inc. とは無関係の個人が運営しています。League of Legends および関連するすべての名称・ロゴは Riot Games, Inc. の商標です。

本サービスは Riot Games 利用規約および開発者ポリシーに従って運営しています。

本ポリシーは予告なく変更される場合があります。変更がある場合は本ページにて通知します。
